About Me

Full Stack Developer

Hello! I'm Yashas M Samrat, a dedicated and versatile Full Stack Developer with a strong foundation in both front-end and back-end technologies. My development journey started with curiosity and quickly evolved into a passion for building intuitive, high-performance web applications that solve real-world problems.

I have experience working with modern tech stacks such as HTML, CSS, JavaScript, React, Node.js, Express.js, MongoDB, and MySQL. On the back end, I’ve built secure APIs, implemented authentication systems, and worked with databases to manage dynamic content. On the front end, I strive to deliver clean, accessible, and responsive interfaces using React and modern CSS frameworks like Tailwind or Bootstrap.

I'm also familiar with version control using Git & GitHub, continuous deployment via GitHub Actions, containerization with Docker, and deploying full-stack apps on platforms like Vercel, Render, or AWS.

I enjoy learning new technologies, contributing to open-source, and refining my problem-solving skills through regular challenges and projects. Whether working solo or collaborating in a team, I bring a detail-oriented mindset, adaptability, and strong communication.

I’m currently looking for opportunities where I can contribute as a full stack developer, grow professionally, and help build amazing digital experiences.
